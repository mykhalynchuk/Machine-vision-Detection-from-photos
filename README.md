# Додаток для виявлення автомобілів на зображеннях

## Опис проєкту
Ця програма призначена для знаходження та графічного виділення автомобілів на зображеннях.

Вона використовує:
OpenCV для обробки зображень  
PyQt6 для графічного інтерфейсу  
Haar Cascade для виявлення об’єктів  

Користувач може завантажити зображення, запустити аналіз і отримати результат із виділеними автомобілями.

## Можливості
- Завантаження зображення з файлової системи  
- Виявлення автомобілів за допомогою Haar Cascade  
- Відображення результату в графічному інтерфейсі  
- Використання багатопотоковості (QThread)  
- Очищення та скидання зображення  

## Використані технології
- Python 3  
- OpenCV  
- PyQt6  
- Haar Cascade Classifier  

## Як користуватися

1. Запустити програму  
2. Натиснути кнопку Load Image  
3. Обрати зображення з файлової системи  
4. Натиснути кнопку Analyze  
5. Переглянути результат із виділеними автомобілями  

## Примітки

- Якість детекції залежить від моделі Haar Cascade  
- Haar Cascade може не завжди точно визначати об’єкти, особливо на складних зображеннях  
- Для кращих результатів рекомендується використовувати сучасні моделі, наприклад YOLO або інші нейронні мережі  
- Використання багатопотоковості (QThread) дозволяє уникнути зависання графічного інтерфейсу під час обробки зображення
